# My personal blog
Home to my personal blog and comments powered by utteranc.es. The build code is open source and free to use. My blog content, name, and likeness are not.

That means you can freely create your own blog using the build scripts and ideas but you must removing my icons, references to my blog, my Google Analytics tag, etc.

I will be creating another repo with a smaller demo that you can fork and use on your own repo.
